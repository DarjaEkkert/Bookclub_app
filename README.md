\# Bookclub App



\## Beschreibung

Eine Webanwendung zur Verwaltung und Bewertung von Büchern für einen persönlichen Buchclub.



\## Features

\- Bücher hinzufügen

\- Bewertung (1–5 Sterne)

\- Schreiben von Rezensionen

\- Bearbeiten und Löschen von Büchern (CRUD)



\## Entwicklung

Die Anwendung wurde iterativ mit Hilfe von KI-Tools entwickelt.

\## Erweiterungsidee

Die Anwendung kann zu einer Multi-User-Plattform erweitert werden, bei der Administratoren Bücher verwalten und Nutzer Bewertungen und Rezensionen hinzufügen.



\## Fazit

Die Anwendung zeigt den vollständigen Entwicklungsprozess von der Idee über die Umsetzung bis zur möglichen Erweiterung.
